
# RNN News Classification

## Project Overview
This project builds a Recurrent Neural Network (RNN) model to classify news articles as **True or fake** using text data.

## Dataset
True and Fake News dataset containing news articles labeled as real or fake.

## Preprocessing
- Text cleaning
- Tokenization
- Padding sequences
- Train-test split

## Model
Recurrent Neural Network built using TensorFlow/Keras.

Architecture:
- Embedding layer
- SimpleRNN / LSTM layer
- Dense output layer with sigmoid activation

## Training
- Optimizer: Adam
- Loss Function: Binary Crossentropy
- Epochs: 10–20

## Evaluation
Model performance evaluated using accuracy on the test dataset.

## Libraries Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- NLTK
